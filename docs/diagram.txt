┌─────────────────────────────────────────────────────────────────────────┐
│                      SIGNAL FLOW (ISO/OSI Layers)                        │
│                                                                         │
│  Layer 7: Application  │ OBD2 (J1979) │ UDS (ISO 14229) │ KWP2000      │
│  Layer 4: Transport    │ ISO 15765-2 (CAN-TP) / Single Frame           │
│  Layer 3: Network      │ 11-bit CAN ID (0x7DF, 0x7E0, 0x7E8)          │
│  Layer 2: Data Link    │ MCP2515 built-in CAN controller                │
│  Layer 1: Physical     │ ISO 11898 (CAN High/Low, 500 kbps)             │
│                                                                         │
│  Bus Configuration:                                                     │
│    CAN_H: 3.5V (recessive) / 2.5V (dominant)                           │
│    CAN_L: 1.5V (recessive) / 2.5V (dominant)                           │
│    Termination: 120Ω resistor at each end                               │
└─────────────────────────────────────────────────────────────────────────┘
